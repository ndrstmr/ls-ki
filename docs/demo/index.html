Live Feature

Eine Klasse = ein neues Feature

Das Tool-Registry-Pattern erlaubt es, zur Laufzeit neue Funktionen hinzuzufügen — ohne Änderungen am Agent-Core.

  • 01QualityCheckTool.php erstellen (implementiert ToolInterface)
  • 02Symfony Autoconfigure erkennt das neue Tool automatisch
  • 03cache:clear → Tool ist in der Registry registriert
  • 04API-Call mit "quality_check": true
  • 05Response enthält jetzt Score + Regelabweichungen
app:translate --quality-check
# Response-Auszug nach Tool-Integration
